Setup

A bullet is fired into a stationary wooden block hanging as a pendulum. The bullet embeds in the block (perfectly inelastic), and the system swings upward. The bullet's initial velocity is unknown — your task is to determine it.

Collision (momentum): mu = (m + M)v
Swing (energy): ½(m + M)v² = (m + M)gh
so v = √(2gh) and u = (m + M)v ⁄ m
(g = 9.8 m/s² ; pendulum length L = 1.8 m)
Reference line: h = 0.0 cm
Set the masses and the (unknown) bullet velocity, set the reference line to the bottom of the block (h = 0), then press Fire.
